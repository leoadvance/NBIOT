# NBIOT

### 开发平台限定
```
IDE         : IAR V7.8
SDK         : nRF51_SDK_9.0.0_2e23562
Code Editor : Source Insight 4.x
```

### 接口说明
```
SPI:  OLED 
IIC:  Sensor 
UART: Debug
```
### 命令说明
- 设置时间<br>
`setTime 17 4 21 5 12 04 32`<br>
设置时间为 2017-04-21 星期五 12:04:32  


### 操作说明
通过键切换显示页面

### 页面
* 页面1: 温度、湿度、tVOC、eCO2
* 页面2: 当前时间，年月日格式以及Unix格式
* 页面3: 当前tVOC baseline, 存储的baseline以及存储时间
